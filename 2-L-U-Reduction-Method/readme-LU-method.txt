En la presente carpeta se encuentra la solucion de sistemas de
ecuaciones lineales 3x3 utilizando el metodo L U.

Sea el sistema M x = b. Encontrar x dado la matriz 3x3 M y el
vector 3x1 b.

El metodo consiste en descomponer la matriz M en el producto de
dos matrices, una triangular superior (U) y otra triangular
inferior (L), tal que M = L U.

Entonces M x =  L U x = b.

Se soluciona entonces el sistema L SolY = b, con SolY = U x.
Al encontrar el vector SolY, se procede a encontrar el vector x

U x = SolY.

En el programa el vector x se escribe como SolX.
